# App Setup

Ensure you are at least using version 15 of node.

- nvm install 14
- nvm use 14
- node -v

Run on IOS:

- expo start --ios

Run on Android:

- expo start --android

Run Development Build of Android to Test on Simulator:

- expo run:android

Run on Web:

- expo start --web
